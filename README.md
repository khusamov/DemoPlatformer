Учебная игра Демо Платформер для Wemos D1 mini
=============================================

Состав оборудования:
- Wemos D1 mini
- TFT Display 1.8 inch ST7735
- PCF8574
- Buzzer с усилителем на биполярном транзисторе
- 8 кнопок
